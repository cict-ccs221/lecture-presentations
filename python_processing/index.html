What is Processing?

  • Coding with Visual Arts/Graphics
  • Written in Java using JOGL (Java OpenGL)
  • Uses OpenGL for accelerated 2D and 3D
  • For GNU/Linux, Mac OS X, Windows, Android, and ARM
  • Can have multiple modes (Java, JavaScript, Python, R...etc)

History

Developers: Casey Reas (Artist, Professor, Developer) & Benjamin Fry (Artist, Psychotherapist, Broadcaster) - 2001

Do anything related to graphics!

	line(x, y, x+20, y-40)
	line(x+10, y, x+30, y-40)
	line(x+20, y, x+40, y-40)

Sketch 6

Interactivity (Mouse) 


def setup():
	size(100, 100)

def draw():
	if (mousePressed == True):
		if (mouseButton == LEFT):
			fill(0)   # Black
		elif (mouseButton == RIGHT):
			fill(255)   # White
	else:
		fill(126)   # Gray

	rect(25, 25, 50, 50)

Sketch 7

Interactivity (Keyboard) 


angle = 0

def setup():
	size(100, 100)
	fill(0)


def draw():
	global angle
	background(204)
	if (keyPressed):
		if ((ord(key) >= 32) and (ord(key) <= 126)):
			# If the key is alphanumeric, use its value as an angle
			angle = (ord(key) - 32) * 3

	arc(50, 50, 66, 66, 0, radians(angle))
