What are Functions?

  • Fragment of Code
  • Can Accept Arguments
  • Can Produce a Result Value
  • Can Affect How Your App Works


# example
len("hello")
# result: 6

What are Methods?

  • Transforms an Object
  • Also Known as the Modifier
  • Always Belongs to an Object


# example
something.method()

All methods are functions, but not all functions are methods!


# a function
function(something)
# a method
something.method()

In Python, a function is defined using the def keyword: 


def my_function():
	print("Hello CICTzens")

After definition, it can be called:


my_function()	
# result: Hello CICTzens

You can also pass parameters like so:


def my_function(a,b,c):
	print(a+b+c)

my_function(1,0,3)

#result:4

...or return a value


def my_function(a,b,c):
	return a+b+c

print(my_function(1,0,3))

#result:4
